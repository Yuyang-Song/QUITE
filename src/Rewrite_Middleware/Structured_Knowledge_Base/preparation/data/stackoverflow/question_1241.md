### Answer ID: 71131321
<p>I had same error days ago. I had to use <code>base64_encode</code> on the user input before sending the post vars to the API REST</p>
<p>Here an example of what I refer</p>
<p>Before</p>
<pre><code>// Vars causing the issue
// User input from Text Area with CKEDITOR JS
$answ = $this-&gt;input-&gt;post(&quot;answ&quot;);
// User input. Just tetxtbox
$question = $this-&gt;input-&gt;post(&quot;question&quot;); 
</code></pre>
<p>After</p>
<pre><code>$answ = base64_encode($this-&gt;input-&gt;post(&quot;answ&quot;)); 
$question = base64_encode($this-&gt;input-&gt;post(&quot;question&quot;));
</code></pre>
<p>Then</p>
<pre><code>postVars = [
        'siteq'=&gt;$question, 
        'answ'=&gt;$answ
    ];
// Sending the postvars by my post method
</code></pre>
<p>It seems to be relationed to apache mod_security</p>
