## Answers
### Answer ID: 76479789
<p>Like others have already commented on your question, do not use all the <code>GRANT</code> statements at the time of creation.</p>
<pre><code>CREATE USER username  IDENTIFIED BY username  ;
-- ROLES
GRANT &quot;CONNECT&quot; TO username  WITH ADMIN OPTION;
ALTER USER username  DEFAULT ROLE &quot;CONNECT&quot;;
</code></pre>
<p>You may want to add:</p>
<pre><code>GRANT UNLIMITED TABLESPACE TO username;
</code></pre>
<p>But certainly not all the other privileges.</p>
